# 🎙️ AI VoiceFinder

Приложение приложение для распознавания речи на базе WhisperX с удобным графическим интерфейсом и расширенными возможностями обработки аудио. Написано в рамках обучения по программе магистратуры в НИЯУ МИФИ.

**Версия**: 1.0.0  


## Возможности

- **Автоматическое распознавание речи**: точная транскрипция с помощью WhisperX
- **Определение спикеров**: встроенная диаризация для разделения говорящих
- **Временные метки слов**: точные метки времени для каждого слова
- **Улучшение звука**: опциональное повышение качества аудио через LavaSR
- **Поддержка видео**: обработка MTS-файлов через FFmpeg
- **Поддержка GPU**: поддержка CUDA для ускоренной обработки

### Системные требования
- **Python**: 3.9+ (рекомендуется 3.11+)
- **CUDA**: 11.8+ (для ускорения на GPU)
- **FFmpeg**: требуется для поддержки MTS-видео
- **ОЗУ**: рекомендуется 8 ГБ и выше
- **GPU**: NVIDIA с поддержкой CUDA (необязательно, есть fallback на CPU)

## 🚀 Быстрый старт

### 1. Клонировать репозиторий
```bash
git clone https://github.com/yourusername/AI-VoiceFinder.git
cd AI-VoiceFinder
```

### 2. Установить зависимости
```bash
# Установить основные библиотеки
pip install -r requirements.txt

# (Опционально) Установить улучшение звука
pip install git+https://github.com/ysharma3501/LavaSR.git

# (Опционально) Установить воспроизведение аудио
pip install sounddevice
```

### 3. Запустить приложение
```bash
python "AI VoiceFinder.py"
```

## 🎯 Использование

1. **Запустите** приложение, выполнив основной скрипт
2. **Выберите аудио- или видеофайл** через проводник
3. **Настройте параметры обработки**:
   - язык
   - размер модели
   - определение спикеров (диаризация)
   - улучшение аудио
4. **Запустите обработку** — приложение покажет прогресс
5. **Просмотрите результаты** — транскрипты сохраняются с метками времени и метками спикеров
6. **Воспроизведение** — опциональное воспроизведение аудио с синхронизированными метками
